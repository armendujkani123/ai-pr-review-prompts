# Code of Conduct

## Our Pledge

We are committed to a respectful, constructive, and harassment-free environment for users and contributors.

## Expected Behavior

- Be respectful and technically constructive
- Focus criticism on code and design, not people
- Help make review conversations useful and inclusive

## Unacceptable Behavior

- Harassment or intimidation
- Personal attacks
- Discriminatory or abusive language
- Publishing private information without permission

## Enforcement

Maintainers may moderate or remove contributions that violate this code of conduct.
